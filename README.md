# Automatic People Counter (LDR + LM358 + CD4026)

Developed a laser–LDR based people counting system using comparator and digital counter logic.

## Objective

To detect and count people passing through an entry point.

## Components Used

- LDR Sensor
- Laser Source
- LM358 Op-Amp (Comparator)
- CD4026 Counter IC
- 7-Segment Display
- Buzzer

## Working Principle

Interruption of laser beam causes change in LDR resistance, triggering comparator output and incrementing digital counter displayed on 7-segment display.

## Testing

- Verified detection reliability
- Tested count increment functionality
- Confirmed buzzer indication on trigger
